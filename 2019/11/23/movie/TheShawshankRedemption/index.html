观影《肖申克的救赎》

发表于 2019-11-23 更新于 2021-03-09 分类于 人是什么东西

简介下故事剧情：

1946年，年青的银行家安迪·杜弗兰在醉酒的情况下持枪在妻子的情夫家门口堵人，等了半夜没等到人，于是安迪决定先回家睡个觉，半路上还把手枪扔到了河里。神奇的是，第二天妻子及情夫被发现在家中双亡。安迪无法证明自己的清白最终被认定为凶手，判处终身监禁。

1947年初，安迪入狱，一个月后，第一次开口说话的安迪找有渠道且刚假释被拒的瑞德买一把六七英寸长的石锤。之后安迪又找瑞德弄一张女星Rita Hayworth的图画。

1949年的春天，因为做一个修缮屋顶的工作时，安迪得到一个展示自己财务技能的机会，帮狱官免税拿到一笔遗产。之后安迪得到典狱长的关注，被安排到图书馆做一些轻松的工作，闲下来的安迪决定搞点事情，扩建图书馆。典狱长自然不会给钱，安迪决定每周给州参议院写信请求拨资金。

1950年的四月，安迪已经为肖申克监狱一半以上的警卫报税。1951年，安迪为所有的警卫报税，包括典狱长。

1951年，在肖申克服刑了50年的图书管理员老布假释获批，得到消息的老布情绪激动，不愿意出狱。用瑞德的话讲，老布在肖申克监狱待了50年，已经被”制度化”（Institutionalized）了，他在狱内还有点用，但出去之后就一无是处。外面的世界对老布来讲是一个全新的世界，出狱后的老布承受着各种艰难不适和担惊受怕。在老布给狱友写了一封道别封信后选择了离开这个世界。

1954年，写了六年每周一封的请求信后，安迪得到了州参议院的答复，拨款200美元及捐赠部分书籍杂物。并不满足的安迪决定之后每周写两封信。

1959年，州参议院决议每年支付五百美元好让安迪封笔。同年，安迪成为典狱长的“私人助理”，以一个虚拟的合伙人账户帮典狱长洗黑钱。

1965年，汤米因盗窃入狱服刑两年。汤米想考取高校文凭找到安迪，安迪决定收汤米为徒教，以教授汤米来打发时间。

1966年，汤米碰巧讲出了安迪妻子死亡的真相，安迪找典狱长希望能帮他翻案，虚假的典狱长为了自己的利益将安迪关了起来。此时得知真相的安迪已经服刑19年。接着典狱长暗中杀害了汤米并威胁安迪继续帮他做账。

知道自己没的选的安迪，决定离开。安迪找瑞德长谈了一番如果能出去的想法，并让瑞德答应，如果出去了一定要去约定的地方取自己埋下的东西。

一个电闪雷鸣的夜晚，安迪带着典狱长的账本逃出了肖申克监狱。十几年来，安迪用女影星的图画做遮挡，在墙壁上用石锤挖出一条一人宽的地洞。成功出逃的安迪将典狱长的恶性公之于众并使用洗钱用的虚假账户去往墨西哥开始新的生活。

服刑四十年的瑞德获批了假释，出狱后的瑞德像老布一样很难面对生活，处于担惊受怕的状态。支撑瑞德的是答应要取安迪埋下的东西。安迪越狱后埋下的是一笔出国资金和一封鼓励瑞德开始新生活的信。

故事的结局两个老朋友在墨西哥的太平洋海边相见。

经典台词：

这个世界忙成一团。

希望是个很危险的东西，希望能让人疯掉，你最好还是认命吧。

我想我只有一个选择，要么忙着生，要么忙着死。

有些鸟儿是关不住的, 因为他们的每一片羽毛都沾着自由的光辉.

只是我回首过往时，那个年轻的犯下重罪的蠢孩子，我想和他谈谈，我想和他谈谈人生道理，告诉他是非善恶，但我不能，因为那个孩子早就不见了，只剩下这位耄耋老人孤身一人。

可怕的是生活在恐惧之中。

如果你已经走了这么远的路，也许你还可以再往前走一点。

希望是件美好的事，也许是世间最美好的东西，而美好的事物永不消逝。

一个自由人开始了一段漫长的旅程，也伴随着未知的结局。
